# Performance Baseline — core derivations (Phase 4.5)

Generated by `scripts/benchmark.mjs` (which runs `packages/core/src/intelligence/baseline.perf.test.ts` under vitest). These are the pure read-model derivations the AI Platform (Phase 5) calls on every request. They are pure functions of their input, so the numbers below are the stable baseline Phase 5 must not regress.

| Derivation | Mean (µs) | p95 (µs) |
| --- | ---: | ---: |
| `intelligence.buildDashboard` | 30.8 | 58.8 |
| `intelligence.attentionItems` | 2.1 | 2.9 |
| `intelligence.computeSignals` | 10.7 | 10.1 |
| `intelligence.buildReviewSnapshot` | 12.2 | 16.4 |
| `resource.buildResourcePortfolio` | 43.3 | 64.5 |
| `resource.buildPortfolio(25)` | 7.7 | 10.9 |

Measured over 5,000 iterations each. Every derivation completes in well under a millisecond, so composition cost is negligible next to DB round-trips — the AI layer can call these freely.
