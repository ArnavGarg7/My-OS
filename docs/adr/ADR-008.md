# ADR-008 — Memory Architecture

- **Status:** Accepted (frozen, Sprint 5.5)
- **Date:** Phase 5.1 / 5.3

## Context

Conversations need continuity and the assistant needs relevant recall, without embedding a vector database or leaking private data.

## Decision

Memory is deterministic: conversation history windows + rolling summaries keep context small; retrieval uses fixed weights + age (no embeddings/vectors). Permanent memory always requires **explicit user confirmation**. The Personal AI Profile ([ADR-003](ADR-003.md)) learns only from accepted feedback and is distinct from memories.

## Consequences

- Recall is reproducible and inspectable; no opaque similarity model.
- Nothing is remembered permanently without consent.
- Works fully offline on the Local provider.
