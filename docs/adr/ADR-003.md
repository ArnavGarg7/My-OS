# ADR-003 — Chief Architecture (Composition, not Duplication)

- **Status:** Accepted (frozen, Sprint 5.5)
- **Date:** Phase 5.2

## Context

The Chief of Staff must answer "what should I do now?" grounded in the whole system, without re-deriving what the deterministic engines already compute.

## Decision

The Chief (`@myos/ai/chief`) **composes read models** — every score/signal arrives pre-computed from its owning module via the server `composer.ts` seam. The pure Chief core imports **no other domain**. It adds only ranking, confidence bands, explainability, provider policy, and a Personal AI Profile that learns solely from accepted feedback.

## Consequences

- The Chief cannot drift from the OS's truth; it re-presents it.
- The single seam (`composer.ts`) is the only place cross-module reads happen.
- The same pattern powers the Intelligence Dashboard and the conversational assistant.
